## Running locally:

```
git clone https://github.com/GenesisEducationKyiv/software-engineering-school-5-0-kvachikk
```

```
cd software-engineering-school-5-0-kvachikk
```

```
npm install
```

```
npm start
```

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Run tests

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Technology Stack
- Node.js with Nest.js
- TypeScript
- PostgreSQL database (hosted on NeonDB)
- Sequelize ORM
